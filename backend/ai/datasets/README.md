# Training Datasets Directory

Store training data for ML models here.

## Data Sources

1. **user_behavior_events** - User interaction logs
2. **activity_submissions** - Completion data
3. **skill_growth_logs** - Skill progression
4. **user_activity_log** - Activity lifecycle data

## Dataset Format

- CSV files for tabular data
- JSON for structured data
- Parquet for large datasets

## Privacy

- Anonymize user data before training
- Remove PII
- Aggregate where possible
