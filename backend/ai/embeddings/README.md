# Embeddings Directory

Store vector embeddings for semantic search and RAG.

## Planned Embeddings

1. **Activity Embeddings**
   - Semantic representations of activities
   - For similarity-based recommendations

2. **Skill Embeddings**
   - Vector representations of skills
   - For skill gap analysis

3. **User Profile Embeddings**
   - Encoded user profiles
   - For personalization

## Models

- Sentence Transformers
- OpenAI Embeddings
- Custom trained embeddings
