# ML Models Directory

Store trained machine learning models here.

## Planned Models

1. **Career Path Predictor**
   - Input: User skills, activity history, engagement metrics
   - Output: Predicted career paths with confidence scores

2. **Activity Recommender**
   - Input: User profile, completion history
   - Output: Ranked list of recommended activities

3. **Skill Gap Analyzer**
   - Input: Current skills, target role
   - Output: Skills to develop with priority

## Model Format

- TensorFlow SavedModel
- PyTorch .pt files
- ONNX for cross-platform compatibility
